# Img-create-add-comment

## 使用說明

1. 點選需要的圖片

2. confirm

3. 雙擊圖片 新增區塊 `export to excel` 輸出成excel檔案

4. 區塊連點可以編輯， 可以拖曳 + 變更大小


[try it](https://tianyili.github.io/img-create-add-comment/)
